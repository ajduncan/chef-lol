# chef-lol #

Installs and configures lol, a mu* style game service.

## Supported Platforms ##

Ubuntu Linux

## Usage

### chef-lol::default

Include `chef-lol` in your node's `run_list`:

```json
{
  "run_list": [
    "recipe[chef-lol::default]"
  ]
}
```
