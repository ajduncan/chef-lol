# chef-lol #

Installs and configures lol, a mu* style game service.

## Supported Platforms ##

Ubuntu Linux

## Usage

### lol::default

Include `lol` in your node's `run_list`:

```json
{
  "run_list": [
    "recipe[lol::default]"
  ]
}
```
